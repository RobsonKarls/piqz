Welcome to Piqz.

This is my personal educational project, work in progress until I run out of ideas.
The main goal is to learn different subjects on API calls, parsing, displaying data with a MVVM pattern and getting used to to DataBinding. Secondary goal is to have a simple, fast and fun app which displays subreddit or multi content.
